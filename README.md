# EjerciciosPaginaPortSwingger
En este repositorio documento detalladamente la resolucion de los laboratorios practicos de PortSwigger Web Security Academy una de las plataformas de seguridad web mas respetadas del sector.

El objetivo de este proyecto es consolidar mis conocimientos tecnicos en auditoria de aplicaciones web crear una base de conocimiento solida para la certificacion eJPTv2 y compartir metodologias claras con la comunidad de ciberseguridad.

Estructura del Repositorio

El repositorio esta organizado por categorias de vulnerabilidades. Dentro de cada carpeta encontraras el reporte paso a paso con su explicacion teorica, la metodologia de explotacion y las capturas de pantalla.

/sqliUnion - Vulnerabilidades de inyeccion SQL explotadas mediante la tecnica UNION.

/commandInjection - Explotacion e inyeccion de comandos en el sistema operativo (OS Command Injection).
